# adamjlovell-digital-services

Personal website for Adam J. Lovell — a freelance web development agency.

## Stack

- Plain HTML, CSS (Tailwind CDN), vanilla JavaScript
- No build tools, no server, no framework
- Static file hosting (GitHub Pages / Netlify / Vercel)

## Pages

| File | Description |
|---|---|
| `index.html` | Homepage / landing page |
| `website-development.html` | Web development service page |
| `reviews.html` | Client reviews & submission form |
| `contact.html` | Contact / lead capture form |
| `admin.html` | PIN-protected review manager |

## JavaScript

| File | Description |
|---|---|
| `js/main.js` | Mobile nav toggle |
| `js/contact.js` | Contact form logic |
| `js/reviews.js` | Fetches & renders live reviews from JSONBlob |
